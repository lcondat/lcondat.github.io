Welcome to Laurent Condat's Image base

The purpose of this database is to provide a large (150) number of images of natural scenes for use in image processing/computer vision research and teaching. The images are free for non-commercial purposes. More precisely, they are distributed under the terms of this Creative Commons license.

 

  

Download the archive file LC_Image_Database.zip containing the color and grayscale versions of the 150 images, as well as a README information file.
